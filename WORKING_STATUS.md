# 日本語プログラミング言語 - 完全動作確認

## 概要
このプロジェクトは日本語の自然言語構文を使ってプログラミングできるライブラリです。

## 動作確認済み機能

### ✅ 基本日本語パーサ (JapaneseParser)
- **変数代入**: `xは10` → `x = 10`
- **算術演算**: `aとb` → `a + b`
- **条件分岐**: `もし条件ならば` → `if condition:`
- **繰り返し**: `回数だけ繰り返す` → `for _ in range(回数):`
- **関数定義**: `関数名は(引数)を返す` → `def 関数名(引数):`
- **文字列処理**: `「文字列」` → `"文字列"`
- **f文字列**: `「{変数}を含む文字列」` → `f"{変数}を含む文字列"`

### ✅ テストスイート
- 10個の包括的テストケースを作成済み
- 全テストケースがパスすることを確認
- 変数、演算、条件、ループ、関数、文字列の各機能をテスト

### ✅ 実行例
動作確認済みのデモプログラム：

1. **基本計算**: 変数と算術演算
2. **成績評価**: 条件分岐
3. **繰り返し**: ループとカウンター
4. **関数**: 関数定義と呼び出し
5. **文字列処理**: 文字列とf文字列

## 使用方法

### 基本的な使い方
```python
import sys
import os
sys.path.insert(0, os.path.join(os.path.dirname(__file__), 'src'))

from parser import JapaneseParser

parser = JapaneseParser()

# 日本語コードを実行
code = """
xは10
yは20
zはxとy
zを表示する
"""

parser.execute(code)
```

### 対応している構文
- 変数宣言: `変数名は値`
- 算術演算: `AとB`, `AからB`, `Aを掛けるB`, `Aで割るB`
- 比較演算: `Aよりも大きいB`, `Aよりも小さいB`, `Aと等しいB`, `Aと異なるB`
- 条件分岐: `もし条件ならば`, `そうでなければ`
- 繰り返し: `回数だけ繰り返す`, `条件の間`
- 関数定義: `関数名は(引数)を返す`
- 出力: `値を表示する`
- 文字列: `「文字列」`
