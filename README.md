# 日本語プログラミング言語モジュール

日本語で書かれたプログラムをPythonコードに自動変換して実行するライブラリです。

## このプログラムは何をするか？

このモジュールを使うと、以下のような日本語プログラムをPythonで実行できます：

```python
# 日本語プログラムの例
xは10
yは20
zはxとy
zを表示する
```

これを自動的に以下のPythonコードに変換して実行します：

```python
x = 10
y = 20
z = x + y
print(z)
# 出力: 30
```

## なでしこ3ベースの設計

このモジュールは日本語の自然な文法構造を使ってプログラミングできるライブラリです。日本語プログラミング言語「なでしこ3」の文法構造を参考にして設計されており、Python初心者でも直感的にプログラミングの基本概念を学習できるように工夫されています。

### なでしこ3との互換性

- **基本文法**: なでしこ3の基本的な文法（変数代入、条件分岐、繰り返しなど）に準拠
- **自然言語構文**: 日本語の助詞（は、と、から、など）を使った自然な表現をサポート
- **教育目的**: Python初心者が日本語でプログラミングの基本概念を学習できるように設計

## 特徴

- **日本語の自然な文法**: `xは10`、`もし条件ならば`、`回数だけ繰り返す`などの直感的な表現でコードを記述可能
- **基本プログラミング機能**: 変数、算術演算、条件分岐、繰り返し、関数定義、文字列処理をサポート
- **自動変換**: 日本語コードをPythonコードに変換して実行
- **なでしこ3互換**: なでしこ3の文法構造に準拠した設計

## 基本的な使い方

```python
from japanese_programming import JapaneseParser

parser = JapaneseParser()

# 簡単な計算機
code = """
数値1は5
数値2は3
合計は数値1と数値2
「合計は{合計}です」を表示する
"""

parser.execute(code)
# 出力: 合計は8です

# 条件分岐の例
code = """
年齢は18
もし年齢よりも大きい20ならば
    「成人です」を表示する
そうでなければ
    「未成年です」を表示する
"""

parser.execute(code)
# 出力: 未成年です

# 繰り返しの例
code = """
回数は3
回数だけ繰り返す
    「こんにちは」を表示する
"""

parser.execute(code)
# 出力: こんにちは
こんにちは
こんにちは

# 関数の例
code = """
関数足し算は(数値1と数値2)を返す
    数値1と数値2を返す

結果は足し算(10と20)
「10と20の合計は{結果}です」を表示する
"""

parser.execute(code)
# 出力: 10と20の合計は30です

parser = JapaneseParser()

# 日本語コードを実行
code = """
xは10
yは20
zはxとy
zを表示する
"""

parser.execute(code)
```

## なでしこ3互換の具体例

### なでしこ3文法との対応
このモジュールはなでしこ3の基本的な文法パターンを採用しています：

```
# なでしこ3の例
Aは10
Bは20
CはAとBを足す

# 本モジュールでの記述
Aは10
Bは20
CはAとB
```

### 教育用言語としての特長
- **直感的な文法**: 日本語の助詞（は、と、から）を使った自然な表現
- **段階的学習**: 変数→演算→条件分岐→繰り返し→関数という順序で学習可能
- **実用性**: 実際のプログラミングで使える基本的な機能を提供

## サポートされている構文

### 変数宣言
```
変数名は値
# なでしこ3互換: 変数名は値
```

### 条件分岐
```
もし条件ならば
    処理
そうでなければ
    処理
# なでしこ3互換: もし条件ならば、そうでなければ
```

### 繰り返し
```
回数だけ繰り返す
    処理

条件の間
    処理
# なでしこ3互換: 回数だけ繰り返す、条件の間
```

### 関数定義
```
関数名は(引数)を返す
    処理
    値を返す
```

### 入出力
```
値を表示する
メッセージを読み込む
```

### 演算子
- `と` - 加算 (+)
- `から` - 減算 (-)
- `を掛ける` - 乗算 (*)
- `で割る` - 除算 (/)
- `よりも大きい` - 大なり (>)
- `よりも小さい` - 小なり (<)
- `と等しい` - 等価 (==)
- `と異なる` - 非等価 (!=)
- `かつ` - 論理積 (and)
- `または` - 論理和 (or)
- `ではない` - 否定 (not)

## 開発の背景

なでしこ3は日本で最も人気のある教育用プログラミング言語の一つです。小学生から大人まで、日本語の自然な文法でプログラミングの基本を学習できます。本モジュールはなでしこ3の設計思想を尊重し、以下の点を重視して開発されました：

1. **自然言語としての日本語**: 助詞や文法構造をそのまま活用
2. **段階的学習**: 簡単な概念から複雑な概念へ
3. **直感的な記述**: プログラミング初心者でも理解しやすい構文
4. **実用性**: 実際のアプリケーション開発で使える機能

## 実行例

```python
from japanese_programming import JapaneseParser

parser = JapaneseParser()

# 基本的な例
code = """
年齢は25
もし年齢よりも大きい18ならば
    "成人です"を表示する
そうでなければ
    "未成年です"を表示する
"""

parser.execute(code)
```

出力：
```
変換されたPythonコード:
age = 25
if age > 18:
    print("成人です")
else:
    print("未成年です")

実行結果:
成人です
```

## 拡張性

このモジュールは拡張可能です。新しいキーワードや構文を追加するには：

1. `keywords.py` の `KEYWORD_MAP` に新しいキーワードを追加
2. `parser.py` の `parse_line` メソッドに新しい構文パターンを追加
