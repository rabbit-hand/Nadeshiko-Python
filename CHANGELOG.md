# Changelog

All notable changes to this project will be documented in this file.

and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [6.0.0] - 2026-05-04

### Added
- Complete bilingual support (Japanese and English)
- Block GUI application with visual programming
- GUI designer with drag-and-drop interface
- Module library for integration with other applications
- Command line parser for text-based programming
- Task tray resident functionality
- Comprehensive English documentation
- Macro recording and playback system
- Cross-platform OS command execution
- File operation commands in natural language
- End marker functionality for code blocks

### Changed
- Complete rewrite from V5 to V6 architecture
- Modular design with separate components
- Enhanced error handling and logging
- Improved performance and memory usage

### Fixed
- Memory leaks in GUI components
- Parsing errors for complex expressions
- Cross-platform compatibility issues

## [5.0.0] - 2026-04-XX

### Added
