### 🎓 教育効果の比較

#### 🐍 Python学習の課題
```python
# Pythonの例 - 子供には難しい
def calculate_area(width, height):
    return width * height

result = calculate_area(5, 4)
print(f"面積: {result}")
```

#### 🌸 なでしこ学習の優しさ
```nadesiko
# なでしこの例 - 子供に優しい
●面積を計算する（幅と高さ）
    幅と高さを掛けて返す

結果は面積を計算する(5と4)
「面積: {結果}」を表示
```
